# Challenge Hospital São Rafael - Sprint 3
## Disciplina: Dynamic Programming
## Professor: André Marques

Este projeto faz parte da Sprint 3 do Challenge da FIAP para o Hospital São Rafael. Ele consiste em uma aplicação em Python voltada para soluções de CRM (Customer Relationship Management) e automação de processos, explorando conceitos de **Dynamic Programming** para otimização de fluxos.

## 🚀 Funcionalidades

- Gerenciamento de dados e interações com pacientes/clientes.
- Processamento automatizado de informações para apoio ao Hospital São Rafael.

## ⚙️ Como Executar

Para rodar a aplicação localmente:

1.  Certifique-se de ter o Python instalado (versão 3.x recomendada).
2.  Clone o repositório:
    ```bash
    git clone https://github.com/CilasMacedo/Sprint-3---DP.git
    ```
3.  Navegue até a pasta do projeto:
    ```bash
    cd Sprint-3---DP
    ```
4.  Execute o script principal (caso disponível):
    ```bash
    python hsr_crm_dp.py
    ```
